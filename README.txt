1) Для настройки работы программы необходимо указать корректные параметры в файле ./src/main/resources/config.properties
Тесты сделаны для браузера FireFox
2) Не корректные данные для логина и пароля записаны в ./src/main/resources/login.csv
3) Корректные данные поставляются логина и пароля поставляются в тесты из dataFromMethod
4) Для запуска позитивного теста через командную строку необходимо ввести clean install test -Dsuite=validLogin
5) Для запуска негативного теста через командную строку необходимо ввести clean install test -Dsuite=invalidLogin
